§2 — Expected Value

The Expected Value Formula

The expected value (EV) of a random variable X is the probability-weighted average of all possible outcomes.

General Definition
E[X] = Σᵢ P(xᵢ) · xᵢ

For a $1 bet on red in European roulette:

Red Bet EV Derivation
E[X] = (18/37)(+1) + (19/37)(−1) = 18/37 − 19/37 = −1/37 ≈ −$0.02703

For a $1 straight-up number bet (pays 35:1):

Single Number Bet EV
E[X] = (1/37)(+35) + (36/37)(−1) = 35/37 − 36/37 = −1/37 ≈ −$0.02703

Key Insight

Every bet in European roulette — regardless of type — has the same expected value of −1/37 per unit wagered. The house edge of 2.703% is invariant. The green zero creates an asymmetry that no combination of bets can neutralize.

§3 — Variance & Standard Deviation

Measuring Uncertainty

While expected value gives the center of a distribution, variance describes how far outcomes spread around that center.

Variance Definition
Var(X) = E[(X − μ)²] = Σᵢ P(xᵢ) · (xᵢ − μ)²
Standard Deviation & Multi-Bet Scaling
σ = √Var(X)
Var(n bets) = n · Var(single)  →  σ(n bets) = √n · σ(single)

Signal-to-Noise Ratio

Expected loss grows as n · |EV|; uncertainty grows only as √n. The ratio |EV_total| / σ_total = √n · |EV|/σ → ∞. The house's advantage becomes increasingly certain as volume increases.
